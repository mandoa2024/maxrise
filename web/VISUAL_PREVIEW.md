## 🎬 动画效果

### Hover 动画
```
卡片/按钮:
  初始: scale(1), shadow-md
  Hover: scale(1), shadow-lg
  过渡: 200ms ease
```

### Focus 动画
```
输入框:
  初始: border-slate-700
  Focus: border-primary + ring-2 ring-primary
  过渡: 200ms ease
```

### 加载动画
```
数据加载时:
  animate-pulse (Tailwind)
  opacity: 0.5 → 1.0
```

## 🎨 玻璃态效果细节

```css
/* 面板背景 */
background: rgba(30, 41, 59, 0.3);  /* slate-800/30 */
backdrop-filter: blur(12px);        /* 背景模糊 */
border: 1px solid rgba(51, 65, 85, 0.5);  /* slate-700/50 */
border-radius: 0.75rem;             /* 12px */
box-shadow: 0 10px 15px rgba(0, 0, 0, 0.1);  /* shadow-xl */
```

## 🌈 渐变应用

### 背景渐变
```css
body {
  background: linear-gradient(
    135deg,
    #0F172A 0%,   /* slate-900 */
    #1E3A8A 50%,  /* blue-900 */
    #0F172A 100%  /* slate-900 */
  );
}
```

### 按钮渐变
```css
.btn-primary {
  background: linear-gradient(
    90deg,
    #1E40AF 0%,   /* primary */
    #3B82F6 100%  /* secondary */
  );
  box-shadow: 0 0 20px rgba(30, 64, 175, 0.5);
}
```

### 卡片渐变
```css
.card {
  background: linear-gradient(
    135deg,
    rgba(30, 41, 59, 0.5) 0%,   /* slate-800/50 */
    rgba(15, 23, 42, 0.5) 100%  /* slate-900/50 */
  );
}
```

## 💡 视觉层次

```
层级 1 (最高): 模态框、下拉菜单
  → z-index: 50
  → shadow-xl

层级 2: Sticky 导航栏
  → z-index: 40
  → shadow-lg

层级 3: 悬浮卡片 (Hover)
  → z-index: 10
  → shadow-lg

层级 4: 普通卡片/面板
  → z-index: 1
  → shadow-md

层级 5 (最低): 背景
  → z-index: 0
  → 渐变背景
```
